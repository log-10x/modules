Defines a Java [OutputStream](https://www.baeldung.com/java-outputstream) to write [TenXObject](https://doc.log10x.com/api/js/#TenXObject) instance and template field values.

OutputStreams provide a simple mechanism for adding custom output destinations 
to write TenXObject/Summary/Template values. 

For an example of a custom OutputStream implementation see [UnixSocketOutputStream](https://github.com/log-10x/pipeline-extensions/blob/main/edge-extensions/src/main/java/com/log10x/ext/edge/output/unix/UnixSocketOutputStream.java#L19)
