Extracts [symbol](https://doc.log10x.com/run/transform/structure/#symbols) values from `.scala` and `.sc` files using [Scalameta](https://scalameta.org).

Enabled by default instead of [ANTLR Scala](https://doc.log10x.com/compile/scanner/antlr/#scala) for faster parsing performance.
