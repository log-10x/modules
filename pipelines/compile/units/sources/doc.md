Defines a list of input folders to recursively [scan](https://doc.log10x.com/compile/scan/) for source/binary files.
Input folders may contain files and sub-folders pulled from [GitHub](https://doc.log10x.com/compile/scan/github/) and [Artifactory](https://doc.log10x.com/compile/pull/artifactory/).
