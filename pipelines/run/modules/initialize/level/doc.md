Enrich TenXObjects with a calculated severity level. 

This module attempts to classify TenXTemplates using specific [symbol](https://doc.log10x.com/run/transform/structure/#symbols) values (e.g.,`Debug`, `Traceback most recent call last`) or [timestamp](https://doc.log10x.com/run/transform/timestamp/) formats that include a severity level (e.g., `'I'MMdd HH:mm:ss.S`).
