Defines language configurations for scanning [symbols](https://doc.log10x.com/run/transform/structure/#symbols) using ANTLR [grammars](https://github.com/antlr/grammars-v4).

Specify a [grammar](#antlrgrammarfile) or [parser](#antlrparserclass) to produce an [AST](https://en.wikipedia.org/wiki/Abstract_syntax_tree) for symbol extraction via configurable [rules](https://doc.log10x.com/compile/scanner/antlr/rules).
