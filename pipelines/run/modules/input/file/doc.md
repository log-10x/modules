Configures a input file path or glob pattern from which to read events to transform into typed [TenXObjects](https://doc.log10x.com/api/js/#TenXObject). 
   
This input does NOT tail its target file/glob. To tail files on disk, use a [forwarder](https://doc.log10x.com/run/input/forwarder) input.
