Scan Go [module](https://go.dev/doc/modules/gomod-ref) files (.mod) for GitHub repository dependencies.

Scanning module files enables the compile pipeline to [pull](https://doc.log10x.com/compile/scan/github/) these repositories and [scan](https://doc.log10x.com/compile/scan/) their source files for symbols to build metadata dictionaries.
