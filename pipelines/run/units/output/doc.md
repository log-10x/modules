Writes [TenXObjects](https://doc.log10x.com/api/js/#TenXObject) and [TenXSummaries](https://doc.log10x.com/api/js/#tenxsummary) to event streams (file, Unix socket, Fluent Bit) and time-series destinations (Prometheus, Datadog). Pipelines support multiple outputs.
